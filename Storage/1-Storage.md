* CloudFront  
* Elastic Block Store (EBS)  
* Storage Gateway  
* Snow family  
* Databases 
  
### S3: 
The S3 (Simple Storage Service) service provides storage for objects (files) 

### EBS: 
Block storage is provided using the Elasic Block Store.
